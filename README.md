# 🐍 Snake Game com Pygame

Um jogo clássico da cobrinha (Snake) desenvolvido em Python utilizando a biblioteca Pygame.

---

## 🎮 Funcionalidades

* Movimento suave baseado em grid
* Crescimento da cobra ao comer comida
* Sistema de pontuação
* Colisão com paredes e com o próprio corpo
* Reset automático ao morrer
* Geração de comida fora do corpo da cobra
* Visual com bordas nos segmentos da cobra

---

## 🛠️ Tecnologias utilizadas

* Python 3
* Pygame

---

## ⚙️ Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/LarissaCarrara/Snake_Game.git
```

### 2. Acesse a pasta do projeto

```bash
cd snake-game
```

### 3. Instale as dependências

```bash
pip install pygame
```



## 🎯 Como jogar

* ⬆️ Seta para cima → mover para cima
* ⬇️ Seta para baixo → mover para baixo
* ⬅️ Seta para esquerda → mover para esquerda
* ➡️ Seta para direita → mover para direita

---

## 🧠 Regras do jogo

* Comer a comida faz a cobra crescer
* Se bater na parede → você perde
* Se colidir com o próprio corpo → você perde
* A pontuação é baseada no tamanho da cobra
